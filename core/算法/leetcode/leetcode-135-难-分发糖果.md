### 题目

老师想给孩子们分发糖果，有 N 个孩子站成了一条直线，老师会根据每个孩子的表现，预先给他们评分。

你需要按照以下要求，帮助老师给这些孩子分发糖果：

- 每个孩子至少分配到 1 个糖果。
- 相邻的孩子中，评分高的孩子必须获得更多的糖果。

那么这样下来，老师至少需要准备多少颗糖果呢？

示例 1:
```
输入: [1,0,2]
输出: 5
解释: 你可以分别给这三个孩子分发 2、1、2 颗糖果。
```

示例 2:
```
输入: [1,2,2]
输出: 4
解释: 你可以分别给这三个孩子分发 1、2、1 颗糖果。
     第三个孩子只得到 1 颗糖果，这已满足上述两个条件。
```

### 题解

```Java
// 贪心的暴力模式 效率很慢 待优化
public int candy(int[] ratings) {
    int[] c = new int[ratings.length];

    c[0] = 1;
    for (int i = 1; i < ratings.length; i++) {
        // 若当前位大于前一位 比前一位多放一个值
        if (ratings[i] > ratings[i - 1]) {
            c[i] = c[i - 1] + 1;
        }
        // 若相等 能少一个少一个
        else if (ratings[i] == ratings[i - 1]) {
            c[i] = 1;
        }
        // 当前位小于前一位
        else {
            c[i] = 1;
            // 若前一位不大于当前位 需要重置
            if (c[i - 1] <= 1) {
                // 依次遍历前面所有的
                for (int j = i - 1; j >= 0; j--) {
                    // 每一位依次比后一位+1
                    c[j] = c[j + 1] + 1;

                    // 直到有一位不小于它的前一位或者j=0 说明重置到头

                    // 0或等于直接pass
                    if (j == 0 || ratings[j] == ratings[j - 1]) {
                        break;
                    }

                    // 当前位大于前一位 取比前后位大一的较大值 并且结束
                    if (ratings[j] > ratings[j - 1]) {
                        c[j] = Math.max(c[j], c[j - 1] + 1);
                        break;
                    }
                }
            }
        }
    }

    int count = 0;
    for (int i : c) {
        count += i;
    }

    return count;
}
```
